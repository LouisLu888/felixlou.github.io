**第一课： 银行躺着赚钱的秘密**

1）银行通过存贷差赚钱的秘密：资金的规模，期限和风险转换

银行3/4净利润是存贷差带来的，剩下1/4分成两大块：

（1）银行投资，比如国债

（2）通道业务费用：银行卖的理财产品。银行在里面只是通道。

这三块时期都依赖于存款能力。银行提供了存贷款之间的**规模转换，期限转换和风险转换**。

2）中国银行的垄断性和高速的经济增长

中国金融的准入门槛很高，银行业垄断。

中国经济高速增长，投资机会多，利润高。

3）新业态：利率市场化和经济增速下滑

准入门槛降低，市场开放竞争

国家推行利率市场化，放开利率管制

银行两极分化，好的很好，但是也有不安全的。

**第二课：货币基金抢了银行的利润吗？**

1）银行间货币市场：银行和我们普通人一样也有资金短缺和盈余的时候，所以经常互相借钱，这种银行间的互相拆借就是银行间货币市场。

它的特点是时间短，规模大，这个特点决定了它的利率比较高。

投向在银行间货币市场的基金就是货币基金。

2）它绕开了银行存款准备金的监管；他让老百姓分享了高收益的银行间的市场利率

货币基金吸银行的血，返回给了老百姓。

3）货币基金收益率：跟着银行间市场利率而变化

钱荒的时候收益率就高，市场宽松钱多的时候，利率就低。

货币基金的投向还包括协议存款（远高于个人存款利率），债券等，所以收益率有可能比单纯的银行间利率高。

**第三课：银行非保本理财靠谱吗？**

1）影子银行是什么？金融创新和监管的博弈

银行是受严格监管的，所以它有绕开监管的冲动。

银行在金融体系中占有绝对的主导地位，因为只有银行能吸收存款，募集资金也比其它机构容易得多。

影子银行就是银行和其他金融机构搞出来一个类银行的产品，一起赚钱。影子银行是金融机构和金融监管之间的猫鼠游戏。

一方面它促使资金流向效率更高的地方。另一方面由于这些金融产品通道很长，又缺乏监管，很容易发生风险事件。

2）银行理财产品的风险

银行只是通道，投资失败银行不用负责任。

3）如何判断理财产品是否靠谱？

（1）注意这个产品的投向

（2）银行是代销的话，说明你不知道产品最后投到了哪里，风险就很大；自营就好很多。

（3）募资方的信息

（4）过高的利率要警惕

**第四课：什么样的货币基金值得购买？**

1）两大原则

（1）挑选散户比较高的货币基金

因为机构型的货币基金对利率变动非常敏感，所以申购赎回都很频繁。这对整体的流动性，安全性都有很大影响。

基金挤兑：funds run

（2）规模适中

规模太大太小收益率低， 因为规模太小协议存款谈判没有话语权，太大的话自己买就会影响整个的价格。

再加上历史业绩比较好，手续费比较低的就是好的基金了。

2）赎回的小窍门

（1）周五不买入，周四不赎回， 因为基金大多是T+1的，而且只计算工作日

（2） 一般月末季末年中和年底的时候，市场资金比较短缺，这时候申购基金收益率比较高

**第五课：货币基金真的没有风险吗？**

互联网金融和传统金融其实没有本质区别，只是在吸款能力和覆盖用户上有一些不同。

理论上货币基金属于基金产品，可能亏损。但实际上安全性相当高，历史上几乎没有亏损的案例。

市场是一个抽象的概念，要理解市场就要理解参与方的行为逻辑，包括政府的逻辑。把政府和市场割裂开，就以偏概全了。

金融市场是个需要严监管的市场，开放一定要谨慎。

市场是个动态演化的过程，其中也包括了监管的动态演化。

市场调节机制的反馈具有滞后性，但是金融行业比较特殊，因为信息不对称和高杠杆，一旦造成后果就非常严重。所以不能过度依靠市场自我调节，宁可监管严一点，步子小一点，也不要出大错。
